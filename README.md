# n8n Automated Reporting Dashboards

Scheduled reporting pipeline that pulls, transforms, and publishes KPI updates.

## Files
- `n8n/workflow.json` importable n8n workflow

## Import
1. Open n8n
2. `Workflows -> Import from File`
3. Select `n8n/workflow.json`
4. Update API URLs and credentials
